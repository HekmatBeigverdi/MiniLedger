# Phase 3 - Authentication and Authorization

## Goal

The goal of Phase 3 is to secure MiniLedger by adding authentication and authorization.

This phase introduces user registration, login, JWT generation, and role-based access control.

---

## What Is Added

- AppUser entity
- ASP.NET Core Identity
- JWT authentication
- Auth DTOs
- Auth service
- Auth controller
- protected endpoints
- role-based authorization

---

## Expected Outcome

By the end of Phase 3, MiniLedger should support secure access to protected operations.
