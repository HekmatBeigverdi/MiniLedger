# Phase 1 - Web API Fundamentals

## Goal
Build the first working version of MiniLedger as a clean ASP.NET Core Web API project.

## Topics Covered
- Solution and project structure
- ASP.NET Core Web API basics
- Entity modeling
- EF Core setup
- CRUD endpoints
- DTOs
- Validation
- Middleware
- Logging
- Swagger

## Modules
- Accounts
- Parties
- Journal Entries
- Journal Entry Lines

## Outcome
A small but structured accounting API with real business validation rules.
